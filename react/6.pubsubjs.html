# 🍏 消息订阅（pubsubjs）

具体的使用方式详见其Github (opens new window)，下面的只是举例如何应用在React中实现父子组件间的相互通信。

# 设计

# 进行消息订阅



 
 
 



class List extends Component {
    componentDidMount(){
        Pubsub.subscribe('update', (msg, data) => {
            console.log(data);
        });
    }
}

# 进行消息发布



 



class Panel extends Component {

        Pubsub.publish('update', [{id: 1}, {id: 2}]);
    }

# 取消消息订阅



 
 
 



 



class List extends Component {
    componentDidMount(){
        this.subscriber = Pubsub.subscribe('update', (msg, data) => {
            console.log(data);
        });
    }

    componentWillUnmount(){
        PubSub.unsubscribe(this.subscriber);
    }
}
