# 数字和字符互转

通过 Number()、parseInt() 和 parseFloat() 都可以。

  • 字符串转数字

一般来说，直接丢到函数里就行，但看起来 Number() 更加万能。

Number("000000");
Number("111.111");
parseInt("222222");
parseFloat("3333.33");

2
3
4
  • 数字转字符串

记得使用括号包裹数字，否则解析器会认为 . 后面是小数点。

(1111).toString();

# 原生类相关

# 时间 Date()

  • 获取时间戳
new Date().getTime();
1
  • 创建某个时间点的 Date 对象
new Date('2020-10-10'); // 传一个时间字符串
new Date(1626346163123); // 传一个时间戳
1
2
  • 获取其他的时间形式
new Date().toLocaleString(); // "2021/7/15下午6:54:07"
new Date().toLocaleDateString() // "2021/7/15"
1
2

# 数字 Math()

  • 小数点上取整
Math.ceil(8.1); // 9
1
  • 小数点下取整
Math.floor(8.1); // 8
1
  • 小数点四舍五入
Math.round(8.1); // 8
Math.round(8.5); // 9
1
2
  • 小数点保留2位小数

先乘100再取整再除以100。

Math.round(4.5678 * 100) / 100
