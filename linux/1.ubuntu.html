cp /etc/apt/sources.list /etc/apt/sources.list.bak

# 安装可爱的 fish (opens new window)

  1. 安装命令
sudo apt install fish -y
1
  1. 设置 fish 为默认终端
chsh -s /usr/bin/fish
1

如果说要配置环境变量，就不能像 bash 那样。fish 有自己的配置文件。

在终端中输入下列语句，使用 nano 打开 fish 的配置文件。

nano ~/.config/fish/config.fish
1

如果这个配置文件不存在也没关系，nano 会自己创建一个新的。

配置一个环境变量的具体语法就是像下面这样，在该配置文件的末尾追加。[ ] 内的内容可以替换成别的软件

set PATH [~/miniconda3/bin] $PATH
1
  1. 附加的主题包 oh-my-fish (opens new window)

在终端输入这些，就会自动下载并安装了。

curl -L https://get.oh-my.fish > install
fish install --path=~/.local/share/omf --config=~/.config/omf
1
2

在这里 (opens new window)可以预览它的所有主题包，我一般是安装 agnoster。

omf install agnoster
1

# 前端开发环境——nvm (opens new window) 与 node (opens new window)

nvm 是一个 node 版本管理器。

  1. 在终端执行以下命令，将自动进行 nvm 的安装：

nvm将被安装到 $HOME/.nvm 目录中

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
1
  1. 让 fish 终端也能用上 nvm

fish-nvm (opens new window) 帮助残障人士使用上 nvm，下面的命令是 omf 配置 nvm 的命令。

omf install https://github.com/fabioantunes/fish-nvm
omf install https://github.com/edc/bass
1
2

这时候在 fish 终端里 nvm --version 应该没问题了。

  1. 安装 nodejs 12.18.3 （LTS）

虽然现在最新的 LTS 版本是 14.17.3，但是管它呢。

nvm install 12.18.3
1

# 静态资源部署 Caddy (opens new window)

具体食用的方式可以点击这里

  1. 在终端运行以下命令以添加存储库
echo "deb [trusted=yes] https://apt.fury.io/caddy/ /" | sudo tee -a /etc/apt/sources.list.d/caddy-fury.list
1
  1. 更新 apt 缓存
sudo apt update
1
  1. 使用以下命令安装 Caddy
sudo apt install caddy
1
